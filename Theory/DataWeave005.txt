//do operator:
A do statement creates a scope in which new variables ,functions, annotations or namespaces can be declared 
and used.. The syntax is similar to a mapping in that it's composed of header and body separated by --- , it's 
header is where all the declarations are defined , and its body is the result of the expression..
syntax:
do{
header
---
body
}.. 

Do statement syntax looks like same as data we've syntax , if we would like to create any variable 
local to the this function, then we can make use of do scope..

%dw 2.0
output application/json

fun fullName(firstName: String, lastName: String): String = do {
	var middleName = "SA"
	---
	firstName ++ " " ++ middleName ++ " " ++ lastName
}
---
{
	"a": fullName("Chinna", "Reddy")
}

//Dataweave core string functions(dw::core):
1.contains:
return true if an input contains given value, false if not..

2.Find:
returns indices of an input that match a specified value..

3.lower:
returns the provided string in lowercase characters..

4.upper:
returns the provided string in uppercase character..

5.matches:
check if an expression matches the entire input string..

6.replace:
performs string replacement..

7.splitBy:
splits string into string array based on value that matches
part of that string...

8.startsWith:
returns true or false depending on whether the input string
starts with matching prefix..

9.endsWith:
returns true if string ends with provided substring , false if not..

10.Trim:
removes any blank spaces from the begining and end of string..

11.isEmpty:
returns true if the given input value is empty , false if not..

12.isBlank:
returns true if the given string is empty(""),completely
composed of whitespaces , or null otherwise, the function returns false..

13.sizeOf:
returns the number of elements in an array.. it returns 0 if the array is empty..

14.typeOf:
returns the primitive data type of value, such as String..

15.namesOf:
returns an array of strings with names of all the keys within the given object..

16.keysOf:
returns an array of keys from key-value pairs within the input object..

17.valuesOf:
returns an array of the values from key-value pairs in an object..

18.indexOf:
returns the index of the first occurence of the specified element in given array , or -1 , if this list 
does not contain the element..

19.lastIndexOf:
returns the index of the last occurence of specified element in given array or -1 if the array does not 
contain element..

20.entriesOf:
returns an array of key-value pairs that describe the key , value and any attributes in the input object..
