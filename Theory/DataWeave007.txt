1. How to find the even numbers from array example: [1,2,3,4,5,6,7,9]
-->
%dw 2.0
output application/json
---
payload filter ((item, index) -> (item mod 2) == 0)

Odd numbers:
%dw 2.0
output application/json
---
payload filter ((item, index) -> (item mod 2) == 1)


%dw 2.0
output application/json
---
payload filter ((item, index) -> (item mod 2) != 0)

2. How to swap firstName and lastName from string example: name = Vitech Talks
-->
%dw 2.0
output application/json
var name =  "Vitech Talks"
var names = name splitBy  " "
---
names[1] ++ " " ++ names[0]
