//Error Handling:
Handling errors are much important when dealing with APIs..
Compile time: Handle by using Anypoint Studio..
Run Time: We need to handle using error handling connector in Mulesoft..

Connectors:
- On error continue: It will not intrupt the entire flow and it will continue with error..
- On error propagate: It will intrupt the flow and throw an error message to use..
- Raise error: If user want to throw custom errors..
